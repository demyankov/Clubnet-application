## Assignment

### Stack:
- **Vite**
- **React**
- **Typescript**
- **Mantine UI**

### Used tools:
- **Firebase**
- **i18next**
- **React Router**
- **Redux Toolkit**

### Implemented features:
- Register with  email and password
- Log in / Log ot
- Reset password
- Private page, accessible only for a logged in user
- Multi language (En, Ru)
- Theme switcher (dark, light)
